# Sims4 Yootz Playthrough

My save in the sims 4 where everything went crazy. It started off in 2016 (before we had pools & toddlers) as me simply exploring the game with 2 adult sims that were roommates. Back then, I was just trying out this whole sims 4 thing, seeing if I wanted to play it instead of the sims 2

At first, I wasn't trying to document what happened, and I didn't intend to put words to what few screenshots I took. But this became my main save file that I found myself returning to over and over. In typical sims fashion, everything got weird and wacky.

**Please note:** This is the **playthrough** version; so filler, randomness, and plot holes are the norm. If you want a cohesive narrative, I'm working on a story version of this in another repo.

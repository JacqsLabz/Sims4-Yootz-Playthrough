# Sims4 Yootz Playthrough

My save in the sims 4 I started in 2016 where everything went crazy. It started off as me exploring the game with 2 adult sims that were roommates (without me trying to document it and without me intending to put words to the screenshots I took), but became my main save file that I found myself returning to. In typical sims fashion, everything got weird.

**Please note:** This is the **playthrough** version; so filler, randomness, and plot holes are the norm. If you want a cohesive story, I'm working on a story version of this in another repo. 

To read this, click whatever part (aka folder) you're on, then on '00-index.md' (should be at the top). Github will open & show it rendered with the text and images together. Or download the project & use your favorite markdown software? Your choice.
